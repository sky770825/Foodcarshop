## 🚀 如何查看效果？

### 1. 啟動開發服務器
```bash
cd lovable優化
npm install     # 首次運行
npm run dev     # 啟動
```

### 2. 打開瀏覽器
訪問 http://localhost:3000

### 3. 測試響應式
- 按 `F12` 打開開發者工具
- 按 `Ctrl + Shift + M` 切換裝置模式
- 測試不同螢幕尺寸
