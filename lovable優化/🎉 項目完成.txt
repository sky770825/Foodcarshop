🚀 如何啟動項目？

方式 1：雙擊 START.bat（最簡單）
  └─ 自動安裝依賴 + 啟動開發服務器

方式 2：手動啟動
  1. 打開命令行（在此資料夾）
  2. 執行：npm install
  3. 執行：npm run dev
  4. 瀏覽器訪問：http://localhost:3000
